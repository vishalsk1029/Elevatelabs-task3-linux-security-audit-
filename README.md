# 🔍 Linux Vulnerability Assessment

This repository contains the results of a basic security audit performed on a Linux Mint system as part of the **Elevate Labs Cybersecurity Internship (Task 3)**.

Instead of searching for vulnerabilities manually, a security auditing tool was used to inspect the operating system and provide recommendations for improving its security posture.

## Tool Used

- Lynis

## Operating System

- Linux Mint
